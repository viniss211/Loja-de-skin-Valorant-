# Loja-de-skin-Valorant-
Aplicação web utilizando HTML,CSS, Bootstrap e Spring boot (java e thymeleaf) de um catálogo de skins do jogo valorant utilizando a IDE do vscode. Para iniciar a aplicação abra o arquivo na IDE indicada e execute o arquivo no caminho loja\src\main\java\br\com\dev\loja LojaApplication.java e use o endereço localhost:8080/index para inicialização
